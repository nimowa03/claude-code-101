# MakeGrowth — 리드마그넷 & 상품 제작

---

## makegrowth.dev = 리드마그넷

랜딩페이지 자체가 리드마그넷.
"수익화 초입 가이드"를 Claude Code 101 스타일로 인터랙티브하게 제공.
이메일 없이 전부 볼 수 있음.
다 보고 "더 받고 싶다" → 이메일 남김.

---

## 이메일 남기면 즉시 받는 것 (래피드)

### ① 수익화 모델 정리본

```
형태: Claude Code 101 스타일 인터랙티브 HTML
내용: 랜딩 5단계를 정리한 별도 페이지
제작: CC101 템플릿 변형. 가장 빠름.
```

### ② 비공개 사례 분석 1~2개

```
형태: 인터랙티브 HTML 또는 노션
내용: 랜딩에 없는 추가 사례
제작: 추가 리서치 필요
```

### ③ 수익화 체크리스트

```
형태: 노션 템플릿
내용: "내 서비스 수익화, 뭐부터 할까?"
  □ 타겟 정의했나?
  □ 수익 모델 정했나?
  □ 가격을 정했나?
  □ 첫 유저를 어디서 데려올 건지 정했나?
  □ 결제 수단을 연결했나?
  □ 런칭 채널을 정했나?
제작: 노션으로 빠르게
```

## 래피드 전달 흐름

```
이메일 입력 (래피드)
  ↓
감사 페이지:
  - 정리본 링크
  - 비공개 사례 분석 링크
  - 체크리스트 링크 (노션)
  - 긱어스 커뮤니티 초대 링크
  ↓
자동 이메일 1통:
  - 위 4개 링크
  - 활용법 안내
```

## 유료 상품

### 첫 유료: 수익화 풀 가이드

```
형태: 인터랙티브 HTML (CC101 변형)
내용: 무료 5단계 → 더 깊은 분석 + 추가 모델 + 워크시트
가격: ₩9,900~19,900 (래피드)
제작: 랜딩 콘텐츠 확장 + CC101 템플릿
```

### 이후: VOD 시리즈

```
형태: 리모션 3~5분 × 5~10편
내용: 케이스별 영상 분석
가격: ₩19,900~29,900 (래피드)
```

### 이후: 케이스 분석 번들

```
형태: 인터랙티브 HTML + 노션
내용: 10~12개 사례 풀 분석
가격: ₩29,000~49,000 (래피드)
```

## 무료 ↔ 유료 관계

```
무료 (makegrowth.dev):         유료 (래피드):
✅ 수익화 초입 가이드 5단계     🔓 풀 가이드 (더 깊은 분석)
✅ 정리본 + 체크리스트          🔓 추가 모델 + 워크시트
✅ 긱어스 커뮤니티             🔓 VOD 시리즈
                              🔓 케이스 분석 번들
```

## 제작 우선순위

```
1순위: 랜딩페이지 완성 (Replit)
2순위: 수익화 모델 정리본 (CC101 변형)
3순위: 수익화 체크리스트 (노션)
4순위: 비공개 사례 분석
5순위: 유료 상품 제작
```
