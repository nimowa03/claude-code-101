# MakeGrowth — 전략

---

## 포지셔닝

**영웅형 크리에이터가 아닙니다. 선생님도 아닙니다.**

수익화에 대한 감이 없는 사람들을 위해
검증된 사례와 인사이트를 대신 찾아서 제공하고,
나 자신도 직접 테스트하면서 같이 배워가는 사람.

| | |
|--|------|
| 빌더 | 직접 서비스를 만들어봄 (상세페이지 빌더 SaaS, 앱인토스) |
| 셀러 | 온라인 셀러. 실제로 물건을 팔아본 경험 |
| 테스터 | 해외 사례에서 발견한 전략을 직접 테스트 중 |
| 커뮤니티 | 같은 고민을 가진 사람들을 모으고 인사이트를 나누는 역할 |

---

## 타겟

```
1. 아이디어만 있는 사람
   — "이런 거 만들어보고 싶은데, 돈이 되긴 하나?"
   
2. 이미 뭐 하나는 만들어본 사람
   — "만들었는데, 이걸 어떻게 팔지?"
   
3. 수익화 초기에 막힌 사람
   — "구독으로 할까? 건당으로 할까? 감이 안 잡힌다."
```

이미 잘하는 사람이 아님. 수익화 초입에 있는 사람들.

사용하는 단어: 사이드 프로젝트, 서비스, 부업, 바이브코딩, 수익화
자제하는 단어: BM, PMF, ARR, CAC

## 수익화 퍼널

```
[유튜브/Threads/긱어스]
     ↓
[makegrowth.dev] — 수익화 초입 가이드 (이메일 없이 체험)
     ↓ "이거 유용하다"
[래피드] — 이메일 수집 → 정리본+비공개사례+체크리스트
     ↓
[긱어스] — 무료 커뮤니티 → 신뢰
     ↓
[유료 상품 — 래피드]
  ├ 풀 가이드 (인터랙티브 HTML) ₩9,900~19,900
  ├ VOD 시리즈 (리모션 3~5분) ₩19,900~29,900
  └ 케이스 분석 번들 ₩29,000~49,000
```

### 각 단계의 역할

```
makegrowth.dev = 가치 먼저 제공 (리드마그넷)
래피드 이메일 = 즉시 정리본/체크리스트 전달
긱어스 = 무료 커뮤니티. 관계+신뢰
유료 상품 = 더 깊은 콘텐츠
```
