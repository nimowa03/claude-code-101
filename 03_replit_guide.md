## 0단계: 디자인 지시 (가장 먼저 전달)

```
프리미엄 SaaS 랜딩페이지 수준으로 만들어줘.
linear.app, vercel.com, supabase.com 같은
최신 다크 테마 랜딩을 참고해서
"이 사이트 돈 주고 만든 거 아니야?" 수준의 퀄리티가 목표야.

활용할 디자인 요소:
- 글래스모피즘 카드 (backdrop-filter blur, 반투명, 미세한 보더)
- 그라데이션 텍스트 (히어로 강조)
- 목업 디자인 (디바이스 프레임 안에 서비스 화면)
- 스크롤형 카드 (Claude Code 101처럼 위→아래로 읽는 구조)
- 진행률 바 (스크롤에 따라 업데이트)
- 호버: 카드 리프트 + 보더 변화 + 그림자 증가
- 스크롤 순차 페이드인 (느리고 부드럽게)
- CTA 버튼 글로우/펄스
- 상단 네비 (고정, 블러 배경)
- 카드마다 다른 강조색

하지 말 것:
- 서비스와 관련 없는 스톡 이미지 금지
- 과한 배경 글로우 금지
- 빽빽함 금지. 여백 충분히

이미지/시각 요소:
- 실제 서비스 스크린샷이나 목업 적극 활용
- 디바이스 프레임(맥북, 브라우저) 안에 화면 넣기
- 섹션 사이에 일러스트, 아이콘, 다이어그램
- 텍스트만 나열하지 말고 시각적으로 몰입감 있게

톤: 깨끗하고 절제된 다크 테마.
"디자이너가 만든 사이트" 느낌으로.
```
